# Pre-Registration — The Behavioral Half-Life

advance* so that outcomes cannot be reinterpreted post hoc. Any deviation is
recorded in an explicit "Deviations" section at the bottom, with date and reason

## 1. Research question

When a model is fine-tuned on the outputs of a previous model that carries a
benign behavioral trait, and this is iterated, does the trait **decay**,
**persist** at a non-zero fixed point, or **amplify** across generations of
fine-tuning? Is there a quantifiable law (a "behavioral half-life")?

This is the *behavioral* analog of model collapse (Shumailov et al., Nature
2024, which studied only distributional/capability degradation) using the
subliminal-learning transmission channel (Cloud et al., Nature 2026).

## 2. Hypotheses (stated before results)

- **H0 (null):** `control_gap_k` (treated minus neutral-teacher trait score) is
  indistinguishable from 0 for all k ≥ 1. No transmission, nothing to track.
- **H1 (decay):** `control_gap_k = A·e^(−k/τ) + c` with c's CI including 0.
  Report **half-life = τ·ln2**.
- **H2 (persistence):** gap converges to a non-zero fixed point (c's CI excludes
  0; no significant trend over the second half of the chain).
- **H3 (amplification):** gap increases across generations (regression slope
  95% CI lower bound > 0).

We do **not** predict which holds; prior work gives opposing intuitions
(Thought Virus prompting chains → decay; Recursive Meta-Distillation theory →
convergence to the trait-free base). Resolving this for *fine-tuning* chains is
the contribution.

## 3. Design (frozen)

- **Two tiers:** (1) toy MNIST auxiliary-logit MLP; (2) small LLMs
  (Llama-3.2-1B primary; Qwen2.5-0.5B/1.5B secondary).
- **Chain:** Gen-0 teacher holds the trait; Gen-k (k≥1) is a fresh copy of the
  **identical base checkpoint B**, fine-tuned on data generated by Gen-(k-1)
  over a semantically-unrelated channel (numbers/code; toy: aux logits).
  *Every generation re-initialises from B* — only the data changes (mitigates
  the Brockers et al. output-head-drift confound).
- **Replicate branches:** B ≥ 3 independent branches per generation; the
  **between-branch** spread is the honest error bar (not within-chain bootstrap
  alone).
- **Controls (mandatory):** neutral-teacher (subtract FT/collapse drift),
  shuffled-data (expect null), cross-base/different-init (expect failure).
  **Defense positive-controls** (divergence-token masking, paraphrase) must
  *suppress* transfer on our rig, replicating Schrodi et al.

## 4. Metrics (frozen)

- **`trait_transfer_score`** — deterministic, **no LLM-as-judge**. LLM tier:
  forced-choice P(pole) and/or first-token/keyword rate on **held-out probes
  containing zero trait tokens**. Toy tier: MNIST test accuracy of the student.
- **`control_gap_k`** = treated − neutral-teacher at generation k (the effect).
- **`utility_retention`** — perplexity delta + task-subset accuracy; tracks the
  model-collapse confound separately from the trait.

## 5. Decision thresholds (frozen — no goalpost movement)

- **Single-hop reproduction gate (Pilot B):** GO iff `control_gap_1` has a
  **95% CI lower bound > δ = 5 percentage points** (BCa bootstrap, 10,000
  resamples). The owl literature reports ≈48pp, so 5pp is deliberately
  conservative. If the gate fails after the diagnosis budget, we ship the
  **negative-result package** (see `docs/` and the paper outline).
- **Toy gate (Pilot A):** student reaches **>50%** MNIST test accuracy vs ~10%
  for the different-init control, reproduced across ≥3 seeds.
- **Dynamics classification:** fit decay / fixed-point / linear models; classify
  per H1–H3 using an **AIC margin ≥ 2** over the flat-line null. A "decay"
  claim additionally requires a significant monotone trend
  (Jonckheere–Terpstra or Mann–Kendall) after FDR correction across generations.
- **CIs:** BCa bootstrap, 95%, 10,000 resamples; between-branch variance used
  for the chain-level band.

## 6. Sample sizes (frozen)

- Eval: ≥ 500 held-out probes per trait per generation (toy: full 10k test set).
- Branches: ≥ 3 (target 5 where compute allows).
- Generations: toy K ≤ 8–10; LLM K = 5–6 local.

## 7. Stopping rules

- Stop a chain early only if `utility_retention` collapses below 50% (model is
  degenerate; record as a collapse event, not a trait result).
- Cap K at the generation where the between-branch CI width first exceeds δ;
  beyond that, late-generation calls are declared underpowered (not "null").

## 8. What would falsify the headline

- Decay law: if removing the neutral-teacher subtraction changes the class, the
  effect is collapse-confound, not trait dynamics → report as such.
- Any positive result that the shuffled-data control *also* reproduces is an
  artifact, not transmission.

## Deviations from this pre-registration

- **2026-06-01 — diagnostic LLM persona-teacher gate.** The committed
  Llama-3.2-1B-4bit persona-teacher gate used 20 forced-choice probes and retained
  imbalanced extracted numeric datasets (484 treated / 143 control) after
  requesting 500 samples. It is therefore reported only as a local diagnostic null
  for that apparatus, not as the fully pre-registered production LLM gate.
- **2026-06-01 — C10 evidence downgrade.** The run log records a leakage-inflation
  observation, but no contaminated-run raw artifact is committed. C10 must remain
  unasserted until that contrast is rerun or recovered as a committed artifact.
- **2026-06-01 — FT-teacher local recovery gate blocked.** The local 1B 8-bit
  FT-teacher positive-control attempt requested 4000 treated samples, but only 13
  yielded clean numeric completions. This is recorded as a channel-blocked
  diagnostic result, not a valid single-hop transfer score.
- **2026-06-03 — Qwen2.5-7B source-faithful addendum before cloud execution.** The next
  production LLM attempt is fixed before any cloud run: `unsloth/Qwen2.5-7B-Instruct`,
  primary benign trait `cat`, secondary planned trait `penguin`, number channel,
  30,000 generated completions per teacher filtered/subsampled to 10,000 clean
  numeric examples per arm, Unsloth/PEFT LoRA rank 8 / alpha 8, neutral-teacher
  and shuffled-number controls, 500 held-out forced-choice probes, BCa 95% CI
  lower bound > +5pp gate, zero trait-token leakage, and a $1,000 budget cap with
  a stop at $800 if no valid single-hop verdict exists. The recursive Qwen chain
  remains blocked unless this single-hop gate clears.
- **2026-06-08 — Mac-local Qwen2.5-7B MLX boundary.** A local MLX approximation
  was run before the source-faithful CUDA/Unsloth gate. It used
  `mlx-community/Qwen2.5-7B-Instruct-4bit`, retained 4000 clean treated/control/
  shuffled number examples with zero trait-token leakage, and produced a negative
  treated-control gap (-4.55pp, 95% CI [-8.03, -1.27]). This is reported as a
  local boundary and does not satisfy the planned production Qwen gate above.
